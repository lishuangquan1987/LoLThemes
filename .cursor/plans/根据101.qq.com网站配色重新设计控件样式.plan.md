# 根据101.qq.com网站配色重新设计控件样式

## 目标

根据截图中的实际配色方案，重新设计所有控件的显示效果，实现严格一致的黑色主题，移除所有动画效果。

## 从CSS文件和截图中提取的精确配色方案

### 主要颜色值（基于CSS文件 `common.css` 和 `rank.css` 的实际值）

#### 背景颜色层级

- **主背景色（最暗）**：`#02151e`（最深背景，用于主容器）
- **次背景色**：`#041822`、`#041c28`（用于面板、侧边栏）
- **卡片/输入框背景色**：`#161a20`、`#191d24`、`#191e25`（用于卡片、输入框、下拉菜单）
- **悬停背景色**：`#1b2128`、`#202831`（用于悬停状态）
- **选中背景色**：`#252c35`（用于选中项背景）

#### 文本颜色

- **主文本颜色**：`#fff5e0`（主要文本，接近白色但带暖色调）
- **次要文本颜色**：`#7e807d`（次要信息、占位符）
- **禁用文本颜色**：`#5e625a`（禁用状态）
- **金色强调文本**：`#b59758`、`#ae9156`（用于强调、链接、选中项文本）
- **青色强调文本**：`#1dbaa9`（用于数据、百分比、选中状态）

#### 边框颜色

- **默认边框**：`#313537`（最常用的边框颜色）
- **深色边框**：`#0d2e3e`、`#24292e`、`#112833`（用于分隔线）
- **金色边框**：`#b59758`（用于选中、焦点状态）
- **浅色边框**：`#9c9c9b`、`#929291`（用于浅色元素）

#### 选中/激活状态颜色

- **选中强调色（青色）**：`#1dbaa9`（用于导航项左侧竖条、标签页下划线、选中文本）
- **选中强调色（蓝色）**：`#40a5e5`、`#4385e9`（用于某些选中状态）
- **选中背景色**：`#041c28`（用于导航项选中背景，带左侧竖条）

#### 状态和强调色

- **错误/警告红色**：`#f32643`、`#f14e4e`、`#e54050`、`#c53644`（用于错误、HOT标签）
- **成功绿色**：`#1dbaa9`（与选中色相同，用于成功状态）
- **金色强调**：`#b59758`、`#9a7f47`（用于按钮、强调元素）
- **蓝色强调**：`#40a5e5`、`#4385e9`（用于链接、某些强调）

#### 特殊用途颜色

- **进度条填充色**：`#1dbaa9`（青色，用于进度条）
- **数据可视化颜色**：
- 红色系：`#d44242`、`#c53644`
- 蓝色系：`#9faafc`、`#4385e9`
- 绿色系：`#a1d586`
- 青色系：`#49aab9`
- 金色系：`#a18b6b`、`#d7a75f`

### 控件样式特点

1. **按钮**：深色背景，白色文本，细边框，无渐变，无发光效果
2. **输入框（搜索框）**：深灰色背景 `#1C242E`，浅灰色占位符文本，无边框或细边框
3. **导航菜单项选中**：左侧有亮蓝色/青色竖条，背景稍浅（纯色，非渐变）
4. **标签页选中**：有下划线或背景色变化，文本变为亮蓝色/青色
5. **卡片**：深色背景 `#1C242E`，细边框（浅灰色）
6. **列表项选中**：纯色背景变化，无明显渐变
7. **进度条**：绿色填充，无渐变
8. **无动画、无复杂渐变、无阴影效果**

### 列表项选中状态实现（左侧竖条）

列表项选中状态需要实现左侧青色竖条效果，这是网站的核心视觉特征：

#### 通用列表项样式

```xml
<!-- ListBoxItem 选中状态样式（通用列表） -->
<Style TargetType="ListBoxItem">
    <!-- 默认状态 -->
    <Setter Property="Background" Value="Transparent"/>
    <Setter Property="Foreground" Value="#fff5e0"/>
    <Setter Property="Padding" Value="8,4"/>
    
    <!-- 选中状态 -->
    <Style.Triggers>
        <Trigger Property="IsSelected" Value="True">
            <!-- 选中背景 -->
            <Setter Property="Background" Value="#041c28"/>
            <!-- 选中文本颜色（青色） -->
            <Setter Property="Foreground" Value="#1dbaa9"/>
            <!-- 左侧竖条：使用 Border 实现 -->
            <Setter Property="Template">
                <Setter.Value>
                    <ControlTemplate TargetType="ListBoxItem">
                        <Border Background="{TemplateBinding Background}"
                                BorderBrush="#1dbaa9"
                                BorderThickness="3,0,0,0">
                            <ContentPresenter Content="{TemplateBinding Content}"
                                            Foreground="{TemplateBinding Foreground}"
                                            Margin="{TemplateBinding Padding}"/>
                        </Border>
                    </ControlTemplate>
                </Setter.Value>
            </Setter>
        </Trigger>
        
        <!-- 悬停状态 -->
        <Trigger Property="IsMouseOver" Value="True">
            <Setter Property="Background" Value="#1b2128"/>
        </Trigger>
    </Style.Triggers>
</Style>
```

## 注意事项

1. **颜色一致性**：所有控件必须使用统一的颜色资源，不能硬编码颜色值
2. **无渐变原则**：所有背景、边框、填充都必须使用纯色（SolidColorBrush），禁止使用渐变
3. **无动画原则**：所有状态切换必须即时完成，不能有任何过渡动画
4. **无阴影原则**：禁止使用 DropShadowEffect、BlurEffect 等视觉效果
5. **选中状态颜色区分**：

   - **主要导航元素**（ListBox、Menu、标准TabControl）：使用青色（`#1dbaa9`）作为选中强调色
     - 背景：`#041c28` + 左侧竖条 `#1dbaa9`（3px宽） + 文本 `#1dbaa9`
   - **筛选器按钮**（SwitchFilter）：使用金色（`#b59758`）作为选中强调色
     - 背景：`#02151e`（保持） + 边框 `#b59758`（1px） + 文本 `#b59758`
   - **输入框焦点**：使用金色（`#b59758`）边框

6. **悬停状态统一**：所有控件的悬停状态使用 `#1b2128` 背景色
7. **边框统一**：

   - 默认边框：`#313537` 或 `#0d2e3e`（深色边框）
   - 焦点/选中边框：`#b59758`（金色，用于筛选器、输入框）或 `#1dbaa9`（青色，用于导航、标签页）
